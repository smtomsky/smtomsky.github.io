Image Main

Stephen M. Tomsky




This Page will be the root for files of the generated Lensing images created by LSST Project programs.
This Page will link to individual directories of images like catalogs of .fits or .jpeg collections of these images


Current plans are for individual links to the images as .jpeg thumbnails here on the site,
while links to the full repositories will be given as links to the github locations, here.

You can always visit my github if you are looking for the main repository.
This site is a work in progress.


List of Features under developement

  • FITS to JPEG Image Converter
  • Image Directory
  • Branching Catalog
  • Online FITS to JPEG Generation/Conversion tool
